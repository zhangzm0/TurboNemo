# 分类标签

技能使用五个标准分类角色。此文件将这些角色映射到本仓库 Issue 追踪器中实际使用的标签字符串。

| mattpocock/skills 中的标签 | 本仓库中的标签 | 含义                                  |
| -------------------------- | -------------- | ------------------------------------- |
| `needs-triage`             | `needs-triage` | 维护者需要评估此 Issue                |
| `needs-info`               | `needs-info`   | 等待报告者提供更多信息                |
| `ready-for-agent`          | `ready-for-agent` | 已完整描述，可供 AFK 代理处理     |
| `ready-for-human`          | `ready-for-human` | 需要人类实现                       |
| `wontfix`                  | `wontfix`      | 不予处理                              |

当技能提到某个角色时（例如"应用 AFK-ready 分类标签"），使用此表中对应的标签字符串。

如需修改，直接编辑右侧列以匹配你实际使用的标签词汇。
