## 代理技能

### 问题追踪

Issue 存放在 GitHub 上 `zhangzm0/TurboNemo` 仓库。外部 PR 不作为处理入口。详见 `docs/agents/issue-tracker.md`。

### 分类标签

五个标准标签：`needs-triage`、`needs-info`、`ready-for-agent`、`ready-for-human`、`wontfix`。详见 `docs/agents/triage-labels.md`。

### 领域文档

单上下文仓库。创建后会在根目录生成一个 `CONTEXT.md`。详见 `docs/agents/domain.md`。
